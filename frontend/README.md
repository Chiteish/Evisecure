# EviSecure - Digital Evidence Locker Frontend

EviSecure is a modern, enterprise-grade AI-powered Digital Evidence Locker web application built for Police Departments, Forensic Laboratories, and High Courts.

## 🚀 Features

- **Cybersecurity & Legal Design System**: Dark mode glassmorphism UI styled with Tailwind CSS inspired by Microsoft Defender & IBM Security QRadar.
- **Hero Image Integration**: Exact hero layout match showcasing the EviSecure digital evidence chain visual.
- **Cryptographic SHA-256 Verification**: Instant hashing display, copy tools, and immutable blockchain ledger anchor simulation.
- **AI Forensic Engine**: Spectral deepfake detection, voice match scoring, and neural image tampering analysis.
- **Role-Based Access Simulation**: Dynamic role selector in command center for Police Officer, Forensic Analyst, Judge, and Administrator.
- **Drag & Drop Evidence Vault**: Real-time upload interface with file integrity validation and progress tracking.
- **Interactive Recharts Analytics**: Real-time storage growth and case threat vector distribution.
- **Court-Ready Chain of Custody Reports**: Immutable transfer logs ready for PDF export and judicial presentation.

---

## 🛠️ Tech Stack

- **Framework**: React.js 18 + Vite + TypeScript
- **Styling**: Tailwind CSS + Custom Glassmorphism Utilities
- **State Management**: Zustand
- **Routing**: React Router DOM v6
- **Analytics & Data Vis**: Recharts
- **Icons**: React Icons (Lucide & Feather Icons)
- **Toast Notifications**: React Hot Toast

## 📦 Setup & Local Development

1. **Clone or Navigate to Project**:
   ```bash
   cd evisecure
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Local Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
